## 题目

基于stm32的麦轮避障小车设计

## 主要任务

利用stm32f103最小系统板+相关硬件模块的方式搭建硬件平台，基于stm32库函数进行c语言编程，最终实现避障小车的实地运行。

## 详细功能描述

### 核心功能：

1. wifi/蓝牙遥控功能实现

2. 实现基于麦克纳姆轮的前后左右运动系统

3. 在小车基本运动功能基础上，增加主动避障功能

### 其他辅助功能：

1. 报警提示功能
2. 速度显示功能（测速）
3. 温度检测显示功能（测温）
额外功能：
在小车避障功能的基础上兼容无线遥控功能，可以通过遥控器随时切换小车运动模式（自由避障模式或者遥控模式）
